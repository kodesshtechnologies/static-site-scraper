# BooksToScrape.com Web Scraper (Playwright)

A professional Python scraper that extracts all book data from https://books.toscrape.com/ using Playwright + BeautifulSoup.

## Features

- Full website pagination
- Extracts book title, price, availability, rating, URL
- Outputs CSV + JSON
- Logging system (logs/scraper.log)
- Modular enterprise-grade structure
- Configurable settings

## Output Files

/output/books.csv  
/output/books.json

## Requirements

pip install -r requirements.txt  
playwright install

## Run

python run.py
